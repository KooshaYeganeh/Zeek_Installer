# Zeek_Installer
A script to install zeek software on all distributions


## Run

```
cd /tmp && wget https://codeload.github.com/KooshaYeganeh/Zeek_Installer/zip/refs/heads/main && unzip main && unzip Zeek_Installer-main && ./install && cd
```

## Supported System

**CentOS**
**Debian11****
**Debian12**
**Fedora38**
**Fedora37**
**openSUSE 15.5**
**openSUSE Leap 15.4**
**Ubuntu20.04**
**Ubuntu22.04**
**Ubuntu23.04**
